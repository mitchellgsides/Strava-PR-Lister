Strava Quick Analysis

Welcome to Strava Quick Analysis

This application is designed as an introductory power analysis platform. Using the app, you'll be able to quickly view and compare your power records from individual activities. The app requires power data from Strava activities. If you don't have a Strava account, or your Strava activities do not have power data, please select "Demo Account" in order to see how the application works.
